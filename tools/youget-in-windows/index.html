Windows 下运行 you-get 获取视频

官网

起因

想下载一个优酷的视频，但找了一圈发现过去的在线解析网站如维棠硕鼠等因为版权原因，都已经不再支持优酷了。接下来又找了其他几个网站，要么需要登录会员要么解析出一个m3u8文件，还需要再找工具来下载。尝试用FFmpeg直接下载m3u8列表却发现格式不对。没办法只好转向you-get。

步骤

you-get从最开始以来一直保持活跃的更新，下载策略随着网站的演进也在不断改善，直至今日仍旧能够正确解析大部分网站的视频。然而，由于主要是面对Linux用户开发的，在Windows下使用存在一点不便，绕了点弯路才运行起来。最开始，找到了release中的Windows可执行文件，但运行后发现版本过低，优酷解析策略已经失效了。遂下载最新版。但它和常见的Python package用法不一样，并非import到自己的代码中或是直接cmd -> Python [some.py]运行，而是另一种模式。话不多说，见如下步骤。

  1. pip3 install you-get

    如果Windows没有安装pip3和Python3，请先自行安装配置

  2. 使用cmd或bash等命令行工具切换到[你的工作目录]下

    例如：cd C:\Users\Administrator\Desktop\WorkDir

  3. 运行you-get.exe [your-url]

    例如：”C:\Program Files\Python36\Scripts\you-get.exe” http://v.youku.com/v_show/id_XNjIzMjU5NzI4.html

    注意！cmd中路径需用引号包围，而参数则不需要。与Linux下的写法有差异。

  4. 等待下载完成

  5. （可选）通过FFmpeg对分段文件进行拼接。可通过命令行或SMMVSplitter等基于FFmpeg的GUI工具进行操作。

Voila! 你得到了解析后的视频。熟练操作似乎比原来用硕鼠还能快些。
