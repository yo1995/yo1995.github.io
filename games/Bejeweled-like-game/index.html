由爱消除乐园引发的几点思考

在这半年莫名其妙的岁月里，有一段挥之不去的阴影始终笼罩着我。它便是天天爱消除——爱消除乐园。

这是一款基于微信小程序的网页小游戏，玩法古朴、反馈温和、无氪金，就仿佛从Bejeweled的年代穿越而来。

这并不是一个爆款小游戏，其上瘾程度远远不如最初的跳一跳、后来的弹一弹、以及无数骗人钱财毁人青春的页游。它更像一门难度适中的工作——每天完成一点，持续从中学习，最终获得许多。

亦可比作一门课程。很多人学习过这门课，少数人以优异的成绩通过，大部分人得过且过，有些人浅尝辄止，不少人半途而废。你如何去上这门课，你如何面对人生中一个小小的阶段，完全取决于自己。上与不上，认真与水过，全凭自己。课就在那里，不偏不倚。

它确实曾是小小的梦魇——每每夜里跟自己较劲，便会无头苍蝇般左突右撞，企图心满意足地进入梦乡。但越玩越精神，最后的结果大多是欲求未满的愤懑。尤其是通关时闪耀的屏幕，在严重损毁视力的同时也影响了脑电波中的梦境。曾有多次在梦中出戏，突然“叮铃”的消除通关音效回响在精彩纷繁的故事中，让人哑然失笑。

或许只能算初级课程，大约是计算机组成原理？比他更难的前辈如汇编原理，不仅任务更多、难度更大、变数更丰，还更难在久经考验后取得不错的成绩。更重要的是，投入了几倍的时间，或许获得的提升却极其有限。人的时间是宝贵的，知识的海洋只取一瓢便已是奢望。将时间浪费在不那么有价值的事情上实在可惜。

可以说，从一月左手单手玩开始，它出现在我生活的每个角落：马桶上、课桌旁、卧榻畔、火车厢……走到哪儿，玩儿到哪儿。曾经在三月底第一次非完美通关后放下了两个月时间，却在后来重操旧业。没办法，我还是有点强迫症的，总希望经手自己的事情能够做到满意为止。成于斯，败于斯。

印象最深的几个场景：

去往四川的慢车铺上，春日的油菜花开的正旺，明黄的花野浸透软泥的芬芳。而我，在爱消除；

高高的朝天门呦，毒辣地大太阳嘞。冒着晒中暑的烈日，伫立在两江河口，我在爱消除；

慵懒的午后，一杯竹叶青，歪倒在藤椅上，我在爱消除。

天天爱消除，天天在消除。回想起来，我仍旧不解自己为何如此执念于一件鲜有吸引力的小事。总感觉自己想证明什么，想澄清什么，但鼓起斗志却砸在棉花上。便用水滴石穿的梦境欺骗自己、麻木自己。哦拜托，大好年华怎能虚度？

爱消除乐园让我明白的几件事：

  • 努力一定有收获，但收获不一定有价值。

  • 日积月累能够产生质变。

  • 倘若遇到跨不过去的坎儿，要么放手对自己宽容些，要么积攒足够的实力碾压它。

  • 实力与运气相伴相随，努力奠定了成功的基础，好运是汗水不世的回报。

  • 世界之理在于物理，物理之源源自概率。看透上帝的骰子，便能窥见宇宙的奥秘。

  • 有时正确的选择比实力更重要。

  • 倘若没有清晰的规划，再多的财富也将消耗殆尽。

  • 人外有人，天外有天。

待续或就此搁笔。
