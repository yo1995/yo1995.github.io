逃离恶魔岛

前言

恶魔岛Alcatraz，是旧金山金门大桥入海口附近的一座监狱岛。臭名昭著的黑帮头子阿尔卡彭就曾关押在这里。当然，他和本文内容没有什么关系。

2023年7月15日晚

23时05分。

辛文问我，明天早上的飞机是几点？我看了一下行程，9时15分起飞，8时45分登机。用谷歌地图的depart at功能，预计从圣何塞到旧金山国际机场的开车时间为36分钟。订好了早晨7时30分出发的闹铃，我收拾好东西准备一早出发。

正文

2023年7月16日早

7时37分。

书包东西全部装好，导航位置也已经设好，一切准备就绪，整装待发。不出意外，我将在8时10分左右还完车，20多分钟后通过安检，坐到D11登机口前富富有余地等待开始登机。Breeze Airways的登机流程非常顺畅，从SBD到SFO的行程中，我花了5分钟停好车，1分钟通过SBD空无一人的单通道安检，3分钟登机坐好。波澜不惊。出SFO的时候，虽然耽误了一点时间，但总体而言机场并不大，从轻轨接驳登机口，需要走的距离不远。一切尽在掌控。😎

7时40分。

开出地库，准备上高速。灵机一动：既然会早到机场20多分钟，不如在去机场的路上顺路再去一趟1 finite loop。7月15日忙里偷闲跑到苹果总部门口的零售店䁖一眼，看到了记忆中的、出现在许多“前iPhone时代”有关苹果公司报导照片里、门口竖着大大的1️⃣的苹果公司老正门。因为库比提诺在圣何塞的西边，所以绕路看一眼并不耽误事儿，我想到。昨天去时，光线条件并不好，人流也比较密集，没来得及在门口拍张照。大周日清晨估计没啥人，便一路开到了“无尽环路1号”。

7时56分。

美美地自拍了好几张苹果总部大楼。上车后注意到拐进苹果停车场的路口也有一个路标。便把车扔在路边，又拍了三张搞怪自拍。🤳

8时03分。

距离登机还有一个多小时呢，我离SFO路程约摸着也就半小时左右。再次灵机一动，不如顺道把湾区朋友所说的“海淀黄庄一条街”，湾区补习班一条街也看看吧。苹果总部的路口就在这条街的北端，而南端是遍布一个个“academy”的加州典型的“court”式平房广场。开过去大概也就五分钟，“学而思”门口照张相一分钟，耽误不了啥事情的。

8时04分。

苹果门口左转的红绿灯时间真长。等了两分钟，面前马路一辆车都没有，红灯却一直不变绿。愠怒。😤

8时10分。

看到了传说中的“Think Academy”，在大门对面找了半晌没有自拍的角度，只好把车停在对面，相机放在车顶拍照。拍了两张角度也不成，我想意思到了就成了，差不多得走了吧。导航到SFO租车还车处，预计到达时间8时41分。

看到这个数字时，我开始有点紧张了。谷歌地图预计到达时间(ETA)向来报喜不报忧，绝大多数时候只会延长不会缩短。脑子里飞速地运算：就算一切完全顺利，还车花2分钟，坐轻轨到航站楼花8分钟，过安检求求情插队花5分钟，狂奔到登机口2分钟。等于8时58分。一般登机口关门是提前15分钟。9时00分。我靠。一脚就把油门踩到了底，可无级变速的Altima只用缓慢的加速度回应我的焦急。🐌

上了高速，情况好像又好了起来。路上车流稀少，速度开到85英里毫无问题。我左手开车，右手开始把车座上的杂物囫囵塞进书包里。想法是到了还车处，车停稳立马拎起包跑路。想着想着，又想到一个问题：车的油箱没加满。能不能再抽出2、3分钟，在路上顺路加个油呢？

如果说前面的决定是一个失误接着又一个误判，将我推到了误机的悬崖边缘；那么下高速去Valero加油的决定，则为我误机的命运盖棺定论。

来到湾区时看了一眼路边油价，Costco的油价相比确实便宜不少，但Valero好像也没贵出太多。所以心里一直盘算着还车前去Valero把油加满，然后再开20英里左右，在油表表显依旧满格的情况下还车，和还车按Hertz的油价补油相比，里外里少说能省2刀油钱。事不宜迟，单手搜了一下离高速最近的Valero加油站，设定了一个途中经停点。

8时21分57秒。

人不该在紧张时鲁莽做决定，更不能有侥幸心理以期冀奇迹的出现。

打印完加油小票，我的SFO Rental Return ETA已经来到了8时49分。还会有奇迹吗？在一切尚未尘埃落定时，不能放弃尝试。或许我能9时05分赶到登机口，恰巧赶上Last Call而侥幸逃脱。要是这样的极限操作都能成功，那什么事情干不成？

前日在背题时，经常遇到Aeronautical Decision-Making Model，缩写为DECIDE

  1. Detect the change has occurred
  2. Estimate the need to counter or react to the change
  3. Choose a desirable outcome
  4. Identify actions that can contain the change
  5. Do the necessary action
  6. Evaluate the effect of the action

越是在情况容不得差错时，越是要小心谨慎行事。每一个步骤都应经过这些预先训练过的决策流程，才能避免更多意外的发生。

一路无话，在车流中寻找更快的路径。好在大家开得都很快，总能跟上85英里时速的别的车后面，防止在最不能被pull的时候被pull。

8时48分。

终于赶到了还车处。就在刚下高速的档口，旁边一辆FPIU高速巡警刷地从最左道驶过，旁边车流纷纷降低车速。还好，我一路超速赶来恰好没有遭遇警车。

8时49分15秒。

神经紧绷地盯着手环上的秒针滴答前进。停车，脚刹，背上包就跑。顺着机场轻轨的指示标，一路电梯上了四楼。恰好，两侧的列车都开着门。想都不用想，直接去人多的那辆一定是对的。

8时52分。

小火车好慢啊，可是我什么也做不了。只好把包放下，把东西再次规整，至少让后面的奔跑更能“人包合一贴地飞行”吧。

周五晚上在大华买的三个面包，今天走时没仔细看，随便抓了一个。现在打开面包袋，才发现零星的霉点茁壮成长。好气哦，但无可奈何。撕掉霉点儿，捡了几口还能吃的部分狼吞虎咽，把剩下的半瓶水一饮而尽。

况且、况且。停车楼、国际航站楼、一号航站楼、二号航……

门打开的一霎那，一道黄色的闪电顺势而出。左手扔水瓶，右手丢面包，动作一气呵成。旅人们还没反应过来时，我已经冲向了安检入口。

9时03分。

一般的机场都会有TSA precheck这种的快速通道。最后的希望，我期待能求情让我快速通关。直奔人少的地方跑去，跟工作人员解释了10秒，未果。径直插队走到安检入口前，望着漫长的人流，我败了，败得很彻底。心有戚戚。

一切都如奇迹般顺利，整个流程直到安检门口几乎一秒都没有多花。可还是败给了异想天开和妄自尊大。

9时05分。

Clear pass的小哥还在耳边叨唠“你花两分钟注册我们的app就让你进去”。冗长的安检队伍乌龟般缓慢前移。交谈声咖啡香组成的明亮的机场背景色变得模糊。我笑笑，这一天还是来了。

我的第一次误机。

GONG!🔔

后续

事后诸葛亮而言，真的期待这一天太久了！🥰

以前说过，当悲剧终成定局，自己反而会无由头地乐观起来——因为事情不会变得更糟了。我知道当自己总是这样挑战极限操作，期待幸运眷顾，总会有一天搞砸。一直就希望搞砸的那一次失误成本别太高。

人生第一次坐飞机直接就是北京去伦敦。当时不懂啊，提前了三个小时就来到机场。也不知道怎么进去，还排着队跑到行李柜台，问清楚是拿着纸质登机牌和证件直接过安检和中国海关就行。背着书包拖着小行李箱，走过”no xx beyond this point”，坐着轻轨来到候机室。等了两小时四十分钟百无聊赖后，和邹汝嘉追跑打闹时，一不小心，把护照，掉进了，窗户旁边的，通风井里。

潘桐的妈妈李老师带队，跑前跑后求爷爷告奶奶才找来维修大哥，把栅栏打开，把护照捡出来。等到一切落定，登机都已经进行一半了。

从这件事中我得到了怎样的经验教训呢？很遗憾，以我异想天开的思维方式，得出的结论是，以后尽量别早到，以免无中生事端。

从此以后，我把出发的等候时间越压越紧。其中几次令我记忆深刻的“有惊无险”时刻：

  • 虹桥火车站高铁关门前5分钟到达验票闸机，3分钟在座位上坐稳。
  • 重庆高铁站过了安检发现身份证保管在爹手里忘了拿，没法直接进站，还有20分钟开车。补办临时身份证明，人工出纸质票，扫码进站火车刚好停稳。
  • 新泽西和同学合住，第二天一早从纽约回罗利。没仔细看出发机场以为是纽瓦克，早上闹铃响了才发现是拉瓜迪亚。紧张打车飞驰上岛过河，赶到登机口时已经开始登机。
  • 又一次纽瓦克去波士顿。这次学乖了，坐了最后赶得上飞机的半小时渡轮离开自由女神，特地打车提前1小时到机场。看时间还多就在安检外吃水果喝咖啡。结果安检排队花了0.6小时，慢悠悠走到登机口等了半晌不见人影，一路小跑到真实登机口时显示Last Call。前台本来已经说不给我们上了，跟机舱通话后还是让我们上了最后一排。

紧张刺激又兴奋，乃当我作为最后一个落座的乘客时的体验。这说明一切时间安排都和预计中一模一样，把规划做到了极致，一分钟都没有浪费。可百密终有一疏，常在河边走，哪有不湿鞋？误火车误飞机的那一天总会到来，我笃信。💪

9时10分。

最终环视，航线的柜台空无一人，起飞前15分钟无法退票。我释然了，终于体会到total loss的痛感。由于SBD机场只有这一班航班，唯一的选择就是转战ONT。幸运的是，另一家“挂逼”航线Frontier类似地经营ONT到SFO的廉航。买周日当天的单程不过130刀。目前损失总计45 + 130 = 175。（最后侥幸变成了105）

在查看Frontier机票的时候，忽然一拍脑门意识到，还车时并未检查后备箱。而后备箱里有一件重要的礼物，我相当肯定，确实落在租车的后备箱里了。爸说想在家里文化墙上挂一些装饰盘子，他自己已经攒了不少，觉得金门大桥比较能够代表美国西海岸。谨遵圣旨买了盘子，生怕压在书包里乱糟糟地压坏了，所以留在后备箱里。摸了摸书包再次确认没拿。这感情好，误机祸兮福所倚。

9时30分。

坐上反方向的轻轨，步伐拖沓地回到租车大楼。换买的机票下午四点才出发，有充足的时间反思总结经验教训。来到停车楼，租的车早已被工作人员收走清洗，柜台大爷让我自己先去找找，不行再去找他。茫茫车海，何处寻一台不起眼的Altima？

运气再次帮了我。沿着靠近电梯的路边走着，翘首四顾心茫然，突然看到眼前的这台车就是黑色、Altima、车上水迹未干。转过来一看车牌儿，得嘞！就儿它。悄么声拉开车门打开后备箱，盘子完好无损地躺在角落里。装包，回航站楼。

9时40-50分。

沿着刚才跑过的路线慢慢走着，吹着口哨尽量漫不经心。电梯来到四楼，发现一件事：居然没法走到轻轨站台！

这才猛地回想起刚才的场景：没仔细看方向指示，还完车径直上了四楼。恰好有一列空驶列车停在站台，两侧车门都打开，我想都没想就穿车而过，恰好赶上对向列车。实际上，对向列车是需要从一层的另一个电梯上去才能坐的。那时那刻老天给足了我天时地利，着实得怪自己决策失误造成了误机“人祸”。

10时20-40分。

等托运的大部队终于散去，柜台的小哥才来得及帮我处理机票事宜。一个花费一百多块和浪费一下午时间的错误，我甘之如饴。

11时00分。

安坐在充电器座位上，开始讲述这一切。你猜的没错，我根本没给电脑带充电器。如果一切计划顺利实施，电脑到家时至少还有15%的电。如果……🪫

尾声

完啦，芭比Q了。误机啦，极限操作失败咯！🤡

2017年和邹宗仁在虹桥的奔跑，险象环生却有惊无险。出发之前我跟他说，信我的好了，绝对能赶上火车。当车堵在路上时间一分一秒地流逝时，尽管一句话没说，我却能读出空气中的犹疑与埋怨。落座后3分钟发车铃响，这才如释重负，攥紧的拳头慢慢松弛下来，重新有说有笑地谈论起旅程。

2019年爸推着登机箱坐到最后一排，匀过气儿来才跟我说，下次还是早点儿去机场吧，赶早不赶晚。

过去的几年，自从学会了时间表规划法，我从过去那个完全自由散漫随性而为的行事方式的我，走向了事事精细安排到个位数分钟的另一个极端的我。九点开会，八点五十九分五十秒绝对不会上线。想法是如果把日间时间精耕细作，留给自己的时间日积月累积少成多，也能做不少事情。确实省了不少时间，也确实看了不少Doug Demuro的奶头乐视频。

这次终于混整了。人生的第一次误机。永远不说永远，但也希望这是最后一次误机。从此以往，希望自己找到了“起个大早”和“赶个晚集”的平衡，藉由这段代价不高但痛感明显的小插曲，能够从容不迫地安排未来人生旅途的轻重缓急。

尾声2

故事的另一半还未补全。顺利登机飞跃加州中央山谷，降落在安大略国际机场时，时间是——

17时21分。

飞机飞行非常顺利，ONT的航班少所以停机接驳也没有耽误时间。唯一的麻烦是机场滑行道正在翻修，所以不得不一路滑到跑道末尾，走最远的出口调头回航站楼。

17时24分。

在起飞前，我就规划好返程的路线。如果想以最“挂逼”的低成本从ONT去SBD，谷歌给我指了条道儿：先坐机场接驳巴士，去库卡蒙格火车站；坐圣贝纳迪诺线火车去圣贝纳迪诺中心车站；从车站坐市政公交去机场旁边的公交站；腿儿着去停车场。公交车两趟加火车一段，总计票价12.75刀。时间虽长，但反正下了飞机以后也没别的干的，就想看看能不能赶上这趟火车。

飞机停稳，舱门打开，前排的旅客非常给力地迅速清空过道。抓起背包，一路小跑，冲出航站楼。Frontier这种廉航最烦人的是停机位一般都在犄角旮旯，离航站楼出口巨远。拿出了跑一千米的肺活量，冲刺到了公交车站，17时26分。

等待了两三分钟，不见任何大巴的踪影。我叹了口南加州炙热的废气——终究，还是错付了💔。公交车时间表上预计到站时间17时25分，我本还幻想自己能像早晨一样神仙保佑诸事顺利，能恰好赶上公交车。现实火辣辣地啪叽在我脸上，欢迎从大城市回到小村庄，没有等待、没有堵车、没有排队的快乐老家。

问题在于，美国的公交车和火车的时间表都是相互配合的。公交车17时25分出站，17时50分到火车站；火车17时55分进站，几分钟后离站。如果没赶上这班公交车，下一班公交就是一小时后的18时25分。而下一趟火车……20时55分？！没曾想，周中密集的火车班次，到了周日晚上，突然如大厂程序员的毛发一样稀疏凋零。完犊子了，我想，又得出血了。

17时29分。

在航站楼外问了一圈有没有人拼车去火车站，灰心丧气地回到网约车打车区。打车去火车站路程大概要15分钟，所以17时30分是我必须做决定的“决策期限”。累了，毁灭吧，干脆直接打车送到SBD停车场吧😩。老子100刀都割肉了💸，在乎你打车的几块钱？Lyft掏出来一看，ONT到SBD居然给我报价48刀！这这这，不过半小时的车程，怎么和我记忆里的价格不太一样呀。

不知哪儿来的念头灵机又一闪，把终点的大头针移出SBD机场的范围，到停车场围墙外。嘿！价格顿时降到了40左右。一不做二不休，干脆把起点也移出机场，价格一下变成了33。想用算法玩儿我？没门儿！老子也是身经百战了。立马下单锁价，奔着机场外的小马路走去。

17时38分。

机场可真鸡贼啊！沿着两点之间直线最短走去，根本就没有人行道可走。

先是被停车场的一道栅栏拦住，成，那我绕过栅栏两端呗；

又被一爿装饰草坪善意地提醒，不管他内套，直接横穿草地朝着目标行军；

最终，面前的一道齐胸铁丝栅栏拦住了我的去路。背后是机场偌大的临客停车场，面前不远就是机场高速出口转弯进小马路的登车目的地。爬雪山过草地、千难万险都过来了，能被你这破铁丝网拦住？踩着一个皮卡的后斗，哐嗤一下子，嗨，老子翻出来了！拜拜嘞你内。

越野跑一公里，省了10刀，赚了，赚啦！

18时08分。

SBD机场停车场西区。老黄蜂静静地停在炽烈的夏日无处躲藏的每一个角落。

早先停车时，我本应该停在东区带棚儿的旅客停车区，却一拍脑门拐到了给长期停车和工作人员停车的无棚西区里。当时想着找个犄角旮旯省得被人看到划车。确实挺犄角旮旯的，转过来一看，车身右侧被花坛喷水器喷得那叫一个花里胡哨乌七八糟。得嘞，中水洗车大成功。

脏了吧唧的右后视镜、前挡右侧泥点子；吱吱嘎嘎需要助力转向液的方向轴；热得怀疑人生的晒了两天的车内温度；晕头转向逃离恶魔岛劫后余生的我。

在一天末尾，夕阳西下，我看着挡风内侧被烈日吞噬的热敏纸上毫无印迹的停车二维码，陷入沉思。他妈的，别告诉老子费尽九牛二虎之力赶到停车场，车子被关在里面开不走了。周日晚上六点半机场早下班了。方圆一公里连只活的鸟儿都看不到。还有奇迹吗？

奇迹真的出现了！车停到自助拦车杆前，还没来得及研究如何付费，杆子“唰”地为我升起。

油门到底，头也不回地逃出生天，逃之夭夭。

尾声3

作为和爸妈视频中大轴讲述的内容，绘声绘色唾沫星子横飞过后，他们帮我总结了技术要点：

  • 现在回头看，“三失足成千古恨”里的三件脑抽蠢事，哪件影响最大？如果没有绕路去学而思并在左转红灯浪费两分钟，且没有下高速去加油的情况下，还是有超过一半的可能能够赶上航班的。当然，回国礼物装饰盘也会因此成为后悔一路的遗憾。🍽
  • 去大城市机场要留足时间。纽瓦克那次是因为我们登机口走错了，所以才被last call，时间其实是够的。在登机截止前，一定要留出至少一个小时就在候机厅里坐稳。💺
  • 租车还车前要想好步骤，检查所有储物空间。不要等到最后交钥匙的时候再手忙脚乱地囫囵查看。✅
  • 爸听到一半问，那只能改签吧？我苦笑，成也SBD败也SBD。这个机场只有一家运块怂的航空公司，每天只有一班。要想下一班，只能等明天。🛬
  • 来往机场的交通方式要考虑好，坐火车留出提前量。🚉
  • 有钱用钱开路，没钱用腿开路。🚶
  • 别老拍脑门，灵机biè老闪。🤦‍♂️
  • 城市，城市在向我呼唤。🏙
